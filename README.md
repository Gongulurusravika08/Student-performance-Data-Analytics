# Student Performance Data Analytics

This project analyzes student academic performance using Python and data analytics techniques.

## Objectives
- Understand how study hours, attendance, and previous grades affect final results.
- Visualize relationships between different academic factors.
- Predict student performance using basic regression modeling.

## Tools & Technologies
- Python
- Google Colab
- Pandas
- Matplotlib
- NumPy

## Dataset
The dataset contains:
- Student ID
- Attendance
- Study Hours
- Previous Grade
- Assignments
- Final Result

## Key Features
- Data cleaning and preprocessing
- Correlation analysis
- Data visualization
- Performance prediction model

## Outcome
The project helps in understanding student learning patterns and provides insights that can be used to improve academic performance.
